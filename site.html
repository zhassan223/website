Projects

LiTerm fpga

we built LiTerm, a fully independent file editor, assembler, and processor on an fpga. write risc-v assembly code directly using a physical keyboard, edit in custom text editor, assemble it into machine code, and execute it ... all without needing an external computer.

we were able to also get it to visualize the results on an hdmi display, where you can see a sorting algorithm work real-time!!!!

probably one of my favorite projects in college. if you want to see more, check out the paper + video!

system verilog • gtkwave • risc-v

exploration of activation-aware quantization for LLMs

explored activation‑aware 4‑bit quantization for large language models—designing two W4A4 schemes (mixed‑precision channel protection and activation‑aware AWQ scaling) to slash memory footprint on LLaMA3‑8B while keeping perplexity within 0.2 ppl of full precision

we ran extensive varying activation bit‑width, quantization group size, and % of protected channels and visualized our results in real‑time charts to see how each tweak impacts model perplexity and compute overhead.
